 # Studly - приложение для трекинга учебной деятельности

Мобильное приложение, созданное для студентов и учащихся, которые стремятся к осознанному и эффективному обучению. Приложение помогает структурировать учебный процесс, ставить цели перед сессией, отслеживать прогресс и закреплять знания с помощью искусственного интеллекта.
